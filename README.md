# PiCrawler

A custom robotics project built using SunFounder's PiCrawler

I made this to learn Python, Git, Bash, robotics fundamentals, and software engineering fundamentals. 

## Hardware

- Raspberry Pi 4 (2GB)
- SunFounder PiCrawler Robot kit

## Project Phases

- Phase 1: Locomotion and Posture Engine (In progress)
- Phase 2: Network Control and Audio Synthesis (Planned)
- Phase 3: Autonomous Logic and Sensory Awareness(Planned)
- Phase 4: Embedded Computer Vision Stack (Planned)
